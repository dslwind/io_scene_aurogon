# Blender28 古剑游戏模型文件导入插件

Version 1.0.8:

- 支持导入模型时，自动将 DDS 转成 PNG 贴图

Version 1.0.6:

模型：

- nif（古剑 1），进度 100%，可用。
- xac、vmesh、model（古剑 2），进度 100%，可用。
- avatar、model（古剑 3），进度 100%，可用。
- avatar、model（古剑 OL），进度 100%，可用。

动作：
- kf（古剑 1），进度 90%，不可用。
- xsm （古剑 2），进度 90%，不可用。
- hka （古剑 3），进度 0%。
- hka （古剑 OL），进度 0%。
